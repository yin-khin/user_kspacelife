React + Vite – Space, Science & International
This project is a React + Vite template adapted for a Space, Science, and International theme. It provides a minimal setup to get React working in Vite with HMR and some ESLint rules — now tailored for exploring cosmic discoveries, scientific innovation, and global collaboration.

Plugins
Currently, two official plugins are available:

@vitejs/plugin-react uses Oxc

@vitejs/plugin-react-swc uses SWC

React Compiler
The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see this documentation.

Expanding ESLint Configuration for Space & Science Projects
If you are developing a production application — especially one involving space exploration data, scientific visualization, or international research collaboration — we recommend using TypeScript with type-aware lint rules. Check out the TS template for information on how to integrate TypeScript and typescript-eslint in your project.
